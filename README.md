# 15113-project2-backend

This repo contains the backend code for my 15-113 Project 2.

Demo Video: https://drive.google.com/file/d/1oHI14Nd43X-9wE_LWNyV_OmsvPjL4hN8/view?usp=sharing

Inspired by The New York Times Connections Game, I created Thumbnail Connections! Instead of having to group words together into 4 categories, the user must group YouTube videos into 4 categories based on the thumbnail images. After the user has successfully groups 4 videos, they must then try to guess the correct category name that connects them. I am most proud of my use of two different external APIs in this project: the OpenAI API and the YouTube Data API. I am also proud of myself for finding a way to use AI creatively and have it generate the 4 random categories each day. I had also never incorporated a database into a technical project, so I am proud of that feature as well. My backend is deployed by Render, and my frontend is deployed by GitHub Pages. To run it locally, the API keys are found in secrets.txt (which ic located in .gitignore). If secrcts.txt is not found, the API keys are then looked for as environment variables.
